# NovaStore SQL Database Project

NovaStore is a SQL Server database project designed for an e-commerce data management scenario. The project includes relational database design, table creation, sample data insertion, SQL reporting queries, view creation, and database backup operations.

## Project Purpose

The purpose of this project is to design and implement a relational database system for an example e-commerce platform named NovaStore. The database stores product, customer, order, and order detail information in a structured relational format.

## Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- T-SQL
- Microsoft Word

## Database Structure

The project contains the following tables:

- Categories
- Products
- Customers
- Orders
- OrderDetails

The database uses Primary Key and Foreign Key relationships to connect product, customer, and order data.

## Features

- Database creation with `CREATE DATABASE`
- Table creation with `CREATE TABLE`
- Primary Key and Foreign Key relationships
- Sample data insertion with `INSERT INTO`
- Reporting queries with `SELECT`
- Data filtering with `WHERE`
- Sorting with `ORDER BY`
- Table joins with `INNER JOIN` and `LEFT JOIN`
- Aggregate reporting with `GROUP BY`, `COUNT`, and `SUM`
- Date analysis with `DATEDIFF`
- View creation with `CREATE VIEW`
- Database backup with `BACKUP DATABASE`

## Sample Reports

The project includes SQL queries for:

- Listing products with stock below 20
- Listing customer order details
- Showing products purchased by a specific customer
- Counting products by category
- Calculating total revenue by customer
- Calculating how many days have passed since each order
